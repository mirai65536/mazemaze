Mazemaze
========
Simple 3D OpenGL first-person maze game.

Depends on libconfig, SFML and SFGUI packages.

![](screenshot.png)

### Downloading and Building:
```
$ git clone --recursive https://github.com/mirai65536/mazemaze.git
$ cd mazemaze
$ cmake .
$ make
```

### Running:
```
$ ./mazemaze
```
